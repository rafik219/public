Simple documentation CentreonAPI

  • Method : get_all_critical_service():

    Definition: Cette méthode retourne en sortie un json contenant la liste des services critiques sur Nagios.

    Usagew: http://1.42.40.191:5000/get_all_critical_service

  • Method : get_all_service_for_host():

    Definition: Cette méthode retourne en sortie un json contenant la liste de tous les services pour un hostname.

    Usage: http://1.42.40.191:5000/get_all_service_for_host/hostname

    Exemple: http://1.42.40.191:5000/get_all_service_for_host/PARALI0511

  • Method : get_all_host():

    Definition: Cette méthode retourne en sortie un json contenant la liste des hosts UP sur Nagios.

    Usage: http://1.42.40.191:5000/get_all_hosts

  • Method : get_grouped_all_host():

    Definition: Cette méthode retourne en sortie un json contenant la liste des hosts UP groupés par POP sur Nagios.

    Usage: http://1.42.40.191:5000/get_grouped_all_hosts
